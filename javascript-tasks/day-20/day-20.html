Interacting with the Document Object Model (DOM)

What is Document Object Model (DOM)?

The DOM represents the structure of an HTML document in a tree-like structure.

Getting Elements

  • Getting Elements by Tag Name

    JavaScript Code: const *variable* = document.getElementByTagName("tag");

  • Getting Elements by Class Name

    const *variable* = document.getElementByClassName("class name");

  • Getting Elements by ID

    const *variable* = document.getElementById("id");

  • Getting Elements by Using querySelector Methods

    const *variable* = document.querSelector('tag');
    const *variable* = document.querSelector('#ID');
    const *variable* = document.querSelector('.class');

    The querySelectorAll method is similar, but it returns a NodeList, which is an array-like object containing all matching elements, rather than just the first match.

    const *variable* = document.querySelectorAll('id/tag/class');
